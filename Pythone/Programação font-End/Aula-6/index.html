Kali Linux é uma distribuição Linux de código aberto baseada em Debian voltada para diversas tarefas de segurança da informação, como testes de penetração, pesquisa de segurança, computação forense e engenharia reversa.

O padrão da indústria

Kali Linux não se trata de ferramentas, nem de sistema operacional.
Linux é uma plataforma.

  • Facilite seu trabalho

    Você pode pegar qualquer Linux e instalar ferramentas de pentesting nele, mas é necessário configurar as ferramentas manualmente e configurá-las. Kali é otimizado para reduzir a quantidade de trabalho, então um profissional pode simplesmente sentar e ir embora.

  • Kali em todos os lugares

    Uma versão de Kali está sempre perto de você, não importa onde você precise. Dispositivos móveis, contêineres, ARM, provedores de nuvem, subsistema Windows para Linux, Máquina virtual pré-construída, imagens de instalação e outros estão disponíveis.

  • Costumização

    Com o uso de metapacotes, otimizados para as tarefas específicas de um profissional de segurança, e um processo de customização ISO altamente acessível e bem documentado, é sempre fácil gerar uma versão otimizada do Kali para suas necessidades específicas.

  • Documentação

    Quer você seja um veterano experiente ou um novato, nossa documentação terá todas as informações que você precisa saber sobre o Kali Linux. Estão disponíveis diversas dicas e “receitas” para ajudar a tirar dúvidas ou solucionar quaisquer problemas. Toda a documentação está aberta, para que você possa contribuir facilmente.

  • Comunidade

    Kali Linux, com sua linhagem BackTrack, possui uma comunidade vibrante e ativa. Existem fóruns Kali ativos, canal IRC, listagens de ferramentas Kali, um sistema aberto de rastreamento de bugs e até sugestões de ferramentas fornecidas pela comunidade.

Card Hover Effect

Dispositivos baratos e de baixa potência, que são ótimos dispositivos para deixar para trás. Laptops modernos baseados em ARM oferecem alta velocidade com bateria de longa duração como plataforma de avaliação.

Bare Metal

Kali de inicialização única ou múltipla, oferecendo controle total sobre o acesso ao hardware (perfeito para WiFi e GPU integrados), permitindo o melhor desempenho.

Cloud

Provedores de hospedagem que possuem Kali Linux pré-instalado, pronto para uso, sem se preocupar em cuidar da infraestrutura.

Containers

Usar Docker ou LXD permite acesso extremamente rápido e fácil a todas as ferramentas do Kali, sem a sobrecarga de uma máquina virtual isolada.

Mobile

Uma plataforma de teste de penetração móvel para dispositivos Android, baseada em Kali Linux. Kali NetHunter consiste em um aplicativo NetHunter, App Store, Kali Container e KeX.

USB

Kali no seu bolso, pronto para usar o Live Boot. Seu Kali, sempre com você, sem alterar o sistema operacional host, além de permitir que você se beneficie do acesso ao hardware.

Virtual Machines

Imagens pré-construídas do VMware e VirtualBox prontas para uso. Permitindo uma instalação do Kali o mais simples possível, com recursos adicionais, como instantâneos, sem alterar o sistema operacional host. E também temos imagens vagabundas.

WSL

O WSL já vem incluído no Windows moderno. Você pode então começar a usar o Kali (e o Win-Kex) sem instalar nenhum software extra.
